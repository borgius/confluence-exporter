# Confluence Exporter (Work in Progress)

Export an Atlassian Confluence space to local Markdown files preserving the hierarchy. Designed for future ingestion into a RAG / knowledge base pipeline.

Status: Planning & scaffolding phase. See `specs/001-i-like-to/quickstart.md` for the intended usage once implementation progresses.

## Development

Install dependencies:

```
npm install
```

Run placeholder CLI (will expand later):

```
npm run dev
```

Run tests (none implemented yet):

```
npm test
```
